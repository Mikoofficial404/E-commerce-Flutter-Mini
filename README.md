## âš™ï¸ Installation & Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/username/ecommerce-app.git
   ```

2. Install dependencies (backend):

   ```bash
   cd backend
   composer install
   ```

3. Install dependencies (frontend dashboard):

   ```bash
   cd dashboard
   npm install
   ```

4. Run Laravel backend:

   ```bash
   php artisan serve
   ```

5. Run React dashboard:

   ```bash
   npm run dev
   ```

6. Run Flutter mobile app:
   ```bash
   flutter run
   ```
